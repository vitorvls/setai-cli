## ğŸ“‚ Main Directories

### `src/` - Source Code

Contains all TypeScript source code:

- **`commands/`**: CLI command implementations (`init.ts`, `config.ts`)
- **`engines/`**: Processing engines (question collection, template processing, validation)
- **`config/`**: Configuration management (API keys, language settings)
- **`services/`**: External services (AI providers)
- **`utils/`**: Utility functions (i18n, output formatting)
- **`types/`**: TypeScript type definitions
- **`__tests__/`**: Unit tests

### `templates/` - File Templates

Contains templates for generating `.cursor` structure files:

- **`.cursor/`**: Base templates in Portuguese (pt-BR)
- **`.cursor.en/`**: English templates
- **`.cursor.es/`**: Spanish templates

Each template directory contains:
- `README.md.template`
- `context/` - Context files
- `rules/` - Rule files
- Other configuration files

### `locales/` - Translations

Contains translation files for i18n:

- **`pt-BR/`**: Portuguese translations
- **`en/`**: English translations
- **`es/`**: Spanish translations

Each locale contains:
- `questions.json` - CLI questions
- `messages.json` - CLI messages
- `validation.json` - Validation messages
- `templates.json` - Template strings

### `docs/` - Documentation

Contains project documentation:

- **`Documentation/`**: User documentation (pt-BR, en, es)
- **`Documentation_4_devs/`**: Technical documentation for developers
- **`.vitepress/`**: VitePress configuration and theme

## ğŸ“„ Key Files

### Configuration Files

- **`package.json`**: Project metadata, dependencies, scripts
- **`tsconfig.json`**: TypeScript compiler configuration
- **`vitest.config.ts`**: Test framework configuration
- **`tsup.config.ts`**: Build tool configuration

### Entry Points

- **`src/index.ts`**: Main CLI entry point
- **`docs/.vitepress/config.ts`**: VitePress documentation configuration
