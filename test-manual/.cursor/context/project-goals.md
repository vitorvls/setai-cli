## 1. Problem Statement

**Problema que estamos resolvendo:**

O projeto FutSalve é uma aplicação web desenvolvida com o objetivo de simplificar e otimizar a organização de partidas de futebol amador, comumente conhecidas como peladas. Através de uma interface intuitiva, o sistema permite que os jogadores agendem partidas, gerenciem a lista de participantes e façam o controle financeiro de maneira eficiente, eliminando a desordem e a complexidade frequentemente associadas a esses eventos. Utilizando tecnologias modernas como TypeScript e Next.js, a solução é construída para ser escalável, segura e fácil de usar.

**Por que este problema importa:**

A organização de peladas de futebol é um desafio recorrente para jogadores amadores, devido à necessidade de coordenar horários, confirmar a participação dos jogadores e gerenciar o compartilhamento de custos. Este problema, embora comum, pode resultar em frustração e desmotivação para os participantes se não for tratado adequadamente. Portanto, há uma necessidade urgente de uma solução que traga eficiência e clareza a esse processo, melhorando a experiência dos jogadores e incentivando a prática regular de esportes.

## 3. Business Objectives

### Primary Goals

- Alcançar mil assinaturas pagas no primeiro ano de operação
- Aumentar a retenção de usuários em 30% dentro dos primeiros seis meses
- Gerar receita mensal recorrente que cubra pelo menos 150% dos custos operacionais até o final do segundo ano
